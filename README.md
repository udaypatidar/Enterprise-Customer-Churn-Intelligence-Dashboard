# Enterprise Customer Churn & Retention Analytics Hub

## 📊 Project Overview
Customer churn is one of the most critical metrics dictating long-term corporate profitability. This end-to-end analytical project focuses on identifying, segmenting, and predicting customer attrition patterns using a comprehensive consumer dataset. 

By applying advanced data visualization and business logic, this Power BI dashboard transforms raw transaction and demographic data into an early-warning retention mechanism. It allows retention teams to isolate precisely *why* and *where* customers are dropping off to deploy targeted marketing interventions.

## 🛠️ Technical Architecture & Tools
- **Analytics Platform:** Microsoft Power BI Desktop
- **Data Architecture:** Highly optimized relational data model leveraging explicit primary/foreign key mappings.
- **ETL Processing:** Power Query utilized for removing null values, typing inconsistencies, and creating custom conditional tracking columns.
- **Calculations Engine:** Advanced DAX measures deployed to evaluate complex business logic, including active user rates, churn percentages, and cohort behaviors.

## 📈 Core Analytical Perspectives & Features
- **Churn Rate Matrix:** High-level executive KPIs capturing total customer loss alongside percentage fluctuations over time.
- **Risk Segmentation:** Advanced filtering breaking down customer attrition by tenure, service contracts, payment methods, and demographic factors.
- **Behavioral Analysis:** Deep-dives into how account usage metrics (e.g., tech support tickets, billing types) correlate directly with the probability of cancellation.
- **Actionable Insights:** Built-in thresholds designed to highlight high-value account segments that are currently entering a high-risk churn window.
